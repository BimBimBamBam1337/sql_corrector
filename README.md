# SQL Schema Corrector

## Описание

В проекте используются две базы данных:

- **prod** — боевая база данных, которую необходимо корректировать
- **test** — тестовый экземпляр

## Запуск окружения

Все команды выполняются через `Makefile`.

1. Поднятие базы данных:

```bash
make up
```

2. Создание mock вресии баз данных:

```bash
make create_tables
```

3. Запуск скрипта:

```bash
make run
```
4. Осталось проверить внесены ли изменения:
```bash
make prod_db
```
Список команд:

### Поднять контейнеры
```bash
make up
```
### Удалить контейнер с его данными
```bash
make down
```
### Остановить контейнеры
```bash
make stop
```
### Зайти в оболочку для prod базы
```bash
make prod_db
```
### Зайти в оболочку для test базы
```bash
make test_db
```
### Проверить запущенные сервесы
```bash
make ps
```
### Создать базы данных для проверки класса
```bash
make create_tables
```
### Запустить скрипт
```bash
make run
```
